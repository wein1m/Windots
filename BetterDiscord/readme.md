## How to apply this BetterDiscord config?

This config is made for the `system24` theme on BetterDiscord.

option 1 ~ file copy  
copy `system24.theme.css` into:
`$env:AppData\BetterDiscord\themes`

Option 2 ~ in-app edit  
Open Discord → settings → BetterDiscord → themes  
install `system24` (if you haven’t already), then click edit and replace its contents with `system24.theme.css`.

andd~ done!
