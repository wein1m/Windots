## How to apply this spicetify configs?
This scheme is made for the `text` theme from Spicetify MarketPlace.
Copy `color.ini` and `user.css` into:
$env:AppData\spicetify\Themes\text\

then, set `color_scheme = DarkNight` inside your config-xpui.ini.

anddd, dont forget to run `spicetify apply` to apply the changes
